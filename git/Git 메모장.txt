[ Git 기본 사용법 ]
    1. Git 다운로드 및 설치
        https://git-scm.com/downloads
    2. Git 환경 설정
        git config --global user.email "egoksg@naver.com"
        git config --global user.name "egoksg"
    3. Workspace로 이동
        cd "Workspace Path"
    4. Git Repository 생성
        git init
    5. Workspace에서 Index로 이동
        Git은 새로운 파일 및 디렉터리를 추적하지 않는(Untracked) 상태로 보고 있다.
        Git이 변경 사항을 추적하기 위해 git add 명령어를 사용 Index에 추가한다.
            git add .
            git add *

    6. 변경된 사항 확인
        git status

    7. Index에서 Local Repository로 이동
        git commit -m "comment"

    8. 원격 Repository 추가
        git remote add origin https://github.com/SungGyunKim/memo.git

    9. 변경이력을 Remote Repository에 있는 master branch에 전송
        git push origin master

[ Git branch create ]
    1. 현재 branch 확인
        git branch  *가 있는 branch가 현재 branch
    
    2. branch 생성
        git branch subdir1
    
    3. branch 이동
        git checkout subdir1

    4. 테스트 파일 생성
        echo "Test Content" > test.txt
        git add test.txt
        git commit -m "test.txt commit"
        git push origin subdir1
    
[ Git branch merge ]
    1. master branch로 이동
        git checkout master
    
    2. master branch와 subdir1 병합
        git merge subdir1

    3. 변경이력을 Remote Repository에 있는 master branch에 전송
        git push origin master

    4. branch 삭제
        git branch -d subdir1
    
    5. branch 확인
        git branch
